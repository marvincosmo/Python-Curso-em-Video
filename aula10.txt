Aula 10 - Estruturas Condicionais

Se antes, os programas que estávamos fazendo seguiam uma linha de comandos única, agora, há "caminhos" no programa que vão ser seguidos, fazendo com que alguns comandos, mesmo que existentes no programa, não sejam usados. Tudo vai depender das CONDIÇÕES dadas pelo usuário ou pela situação.

A grande maioria dos programas possuem condições.

Para cada condição, existe uma representação estruturada, ou representação indentada.

Ex: 
carro.siga()
se carro.esquerda()
	carro.siga()
	carro.direita()
	carro.siga()
	carro.direita()
	carro.esquerda()
	carro.siga()
	carro.direita()
	carro.siga()
senão
	carro.siga()
	carro.esquerda()
	carro.siga()
	carro.esquerda()
	carro.siga()
carro.pare()

O bloco associado ao SE é chamado de bloco verdadeiro. O associado ao SENÃO é chamado de bloco falso.

Os comandos condicionais, para os casos acima, são:

se -> if

senão -> else

OBS: NUNCA ESQUEÇA de colocar os dois pontos depois do comando condicional!

Ex:
if carro.esquerda:
	bloco True
else:
	bloco False

Ex:
tempo = int(input('Quantos anos tem seu carro? '))
if tempo <= 3:
	print('Carro novo')
else:
	print('Carro velho')
print('--FIM--')

Essa estrutura condicional funciona para situações binárias, situações em que há duas condições, ou seja, em que há dois fluxos de comandos, ou dois algoritmos dentro de um só.

Existe outra forma de expor a linha de comandos condicionais, de uma forma com menos linhas, ou uma CONDIÇÃO SIMPLIFICADA.

Seguindo o exemplo citado anteriormente:

tempo = int(input('Quantos anos tem seu carro? '))
print('Carro novo' if tempo <=3 else 'Carro velho')
print('--FIM--')

Esta forma serve para situações de condições mais simples, mas é menos compreensível que sua versão não simplificada.
